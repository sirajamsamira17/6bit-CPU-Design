Build CPU based on following requirements:
1. Word Size of CPU =6
2. ALU Operations =ADD,NOT,SHL
3. Register Number =6
4. Size of RAM =8
5. Word size of ISA and RAM =18
6. CPU Instructions =Register mode, Immediate Mode, JMP, JL
